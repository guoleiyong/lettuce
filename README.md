STMCEN: 生菜健康诊断的时空多模态因果可解释方法

 📋 项目简介

本项目实现了论文《生菜健康诊断的时空多模态因果可解释方法》中提出的STMCEN模型。该模型通过双向跨模态因果注意力机制和门控时空融合模块，实现了对生菜健康状态的准确、早期、可解释诊断。

## 🚀 快速开始

 1. 环境配置

# 创建虚拟环境（推荐）
python -m venv venv

# 激活虚拟环境
# Windows:
venv\Scripts\activate
# Linux/Mac:
source venv/bin/activate

# 安装依赖
pip install -r requirements.txt
